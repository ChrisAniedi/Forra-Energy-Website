# public/

Static files served at the site root.

## Header logo

Save the ForraEnergy header logo here as **`forra-logo.png`** (this exact name).
It will appear automatically in the top navbar at the correct height, with its
aspect ratio preserved (never stretched or distorted).

- Path: `public/forra-logo.png`  →  served at `/forra-logo.png`
- Prefer a transparent-background PNG (or SVG — if SVG, name it `forra-logo.svg`
  and tell the assistant to update the one `src` line).
- Recommended: at least ~600px wide for a crisp render on high-DPI screens.

If the file is missing, the navbar falls back to the drawn logo so nothing breaks.
